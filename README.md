# Job-Research-Interview-Agent

一个面向求职场景的 Agent 应用，围绕岗位 JD，公司信息，面试主题与本地资料，自动完成任务规划、公开信息搜索，本地知识检索，会话记忆整合与面试准备报告生成，并通过 SSE 实时反馈执行进度

## Input

- **岗位 JD**：提供目标岗位的职位描述，包含职责、要求等信息
- **公司信息**：提供目标公司的基本情况、文化、业务等信息
- **面试主题**：提供面试的主要内容或方向，如技术问题、行为问题等
- **本地资料**：提供与面试相关的本地文档、笔记等资料
- **用户补充说明**： 可选
  
## Output

- 调研规划步骤
- 搜索检索中间结果
- 结构化面试准备报错
- 来源引用
- SSE 实时反馈进度查询

## Core Modules
- Planner
- Search
- RAG
- Memory
- Report
- SSE
